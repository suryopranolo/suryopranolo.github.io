Pertemuan 3: Membuat Algoritma Sederhana

Tujuan Pembelajaran

  • Mengerti konsep algoritma dalam berpikir komputasional.
  • Mampu membuat algoritma sederhana untuk masalah nyata.
  • Mengimplementasikan algoritma menggunakan bahasa C++.

Materi

1. Pengertian Algoritma

Algoritma adalah urutan langkah-langkah logis untuk menyelesaikan suatu masalah atau tugas. Dalam berpikir komputasional, algoritma membantu memecahkan masalah secara sistematis sehingga bisa diimplementasikan dengan bahasa pemrograman.

2. Langkah-Langkah Membuat Algoritma

  • Identifikasi masalah.
  • Tentukan input dan output.
  • Buat langkah-langkah penyelesaian secara logis.
  • Uji algoritma dengan contoh nyata.

3. Contoh Algoritma: Menghitung Rata-Rata Nilai Siswa

  • Masukkan jumlah siswa dan nilai masing-masing.
  • Jumlahkan semua nilai.
  • Hitung rata-rata = jumlah / jumlah siswa.
  • Tampilkan hasil rata-rata.

Implementasi C++:

#include <iostream> using namespace std; int main() { int n; double jumlah = 0, nilai, rata_rata; cout << "Masukkan jumlah siswa: "; cin >> n; for(int i = 1; i <= n; i++) { cout << "Masukkan nilai siswa ke-" << i << ": "; cin >> nilai; jumlah += nilai; } rata_rata = jumlah / n; cout << "Rata-rata nilai kelas adalah: " << rata_rata << endl; return 0; }

4. Contoh Algoritma: Menentukan Bilangan Genap atau Ganjil

  • Masukkan bilangan.
  • Cek apakah bilangan MOD 2 = 0 → Genap, jika tidak → Ganjil.
  • Tampilkan hasil.

Implementasi C++:

#include <iostream> using namespace std; int main() { int bilangan; cout << "Masukkan bilangan: "; cin >> bilangan; if(bilangan % 2 == 0) { cout << bilangan << " adalah bilangan Genap." << endl; } else { cout << bilangan << " adalah bilangan Ganjil." << endl; } return 0; }
