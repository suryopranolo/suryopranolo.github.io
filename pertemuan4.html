Tujuan Pembelajaran

  • Siswa memahami pengertian Flowchart dan Pseudocode.
  • Siswa mampu membuat Flowchart dan Pseudocode sederhana untuk menyelesaikan masalah.

Apa itu Flowchart?

Flowchart adalah diagram yang menggambarkan langkah-langkah logis dari suatu proses atau algoritma dengan menggunakan simbol-simbol tertentu.

Simbol Umum Flowchart:

  • Oval: Awal / Akhir
  • Persegi Panjang: Proses
  • Belah Ketupat: Percabangan (Decision)
  • Panah: Alur Proses

Apa itu Pseudocode?

Pseudocode adalah deskripsi algoritma menggunakan bahasa sederhana yang menyerupai bahasa pemrograman, tetapi tidak mengikuti aturan sintaks secara ketat.

Contoh Kasus: Hitung Rata-rata Nilai

Flowchart: (Diagram dapat Anda buat dengan simbol-simbol standar seperti Oval, Persegi Panjang, dll.)

Pseudocode: 
START
  Input n
  total ← 0
  FOR i ← 1 TO n DO
    Input nilai
    total ← total + nilai
  END FOR
  rata ← total / n
  Output rata
END

Manfaat Flowchart & Pseudocode

  • Membantu memvisualisasikan alur algoritma.
  • Mudah dikomunikasikan dan dipahami.
  • Mengurangi kesalahan sebelum diimplementasikan ke bahasa pemrograman.
